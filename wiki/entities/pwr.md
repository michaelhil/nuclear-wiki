# Pressurised Water Reactor (PWR)

A generic PWR is used in Scenarios 2-5, 7, and 9 of Report 4. No specific licensed design is modelled. The PWR provides the operational context for most of the report series' worked examples, including the running RCS temperature anomaly scenario that threads through Report 6's [[capability-gradient]].

Key systems referenced: Reactor Coolant System (RCS), Emergency Core Cooling System (ECCS), containment, reactor protection system, instrumentation (RTDs, pressuriser level, containment radiation monitors).
